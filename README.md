# Amethyst
Príbehová 2D platformer hra
Hlavne pre jedného hráča, v prípade, že budem mať dosť času, pridám lokálny multiplayer - co-op
Hra sa bude skladať z väčšich levelov (tieto levely budú tvorené v Unity level editor)
Hra bude ľahko editovateľná hráčmi, resp. módovateľná
Hra bude prevžne party hra (myslená na zábavu a odpočinok), ale primerane ťažká
